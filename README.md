My portfolio website built using nextJS

![image](main_page_ss.png)

- To run this repository locally
```bash
1. cd portfolio
2. npm i
   #or
   yarn install
3. npm run dev
   #or
   yarn dev
```
Tech stack :
 - TailwindCSS
 - NextJS
 - ReactJs
